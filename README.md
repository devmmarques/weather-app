# weather-app
Weather iOS Project.

## How to install

Clone this project and then:

```bash
#  Access the root project folder.
cd ./weather-app

# Install cocoapods
sudo gem install cocoapods

# Install cocoapods dependencies
pod install
```

# Supported Versions
The minimum version required to run this app is **iOS 13.0.**
